
# Printout Style Guide

This document defines the visual standards for receipt slips generated by the MDWprint application.

## Global Settings
*   **Font Family:** Sans-serif (System default: Arial, Helvetica, Roboto).
*   **Text Color:** Black (`#000000`) - Strictly enforced for thermal printing.
*   **Alignment:** Center (default).
*   **Paragraph Spacing:** 0px (`margin: 0`, `padding: 0`).
*   **Print Margins:** 0px (No whitespace at top of paper).

## Layout & Dimensions
*   **Print Width:** Auto (adapts to thermal paper width, typically 80mm).
*   **Side Padding:** 30px (Left and Right) to ensure text does not touch edges.
*   **Cut Lines (10x Mode):** 1px dashed black border at the bottom of each slip.
*   **Footer:** "-MDW-" centered at the bottom, Font Size: 4px.

## Typography Hierarchy

### 1. Standard Contact Slips (Retention, Aquatics)
*   **Title:**
    *   Size: 24px
    *   Weight: 700 (Bold)
    *   Margin Bottom: 4px
*   **Body Text (Phone/Email):**
    *   Size: 20px
    *   Weight: 400 (Normal)
    *   Margin Bottom: 4px (between lines)

### 2. Code Slips (Changing Rooms, Sauna)
*   **Title:**
    *   Size: 24px
    *   Weight: 700 (Bold)
    *   Margin Bottom: 8px
*   **Code:**
    *   Size: 32px
    *   Weight: 700 (Bold)

### 3. Timetable Slips (Gym Times, OAP Swims, Rhymetime)
*   **Title:**
    *   Size: 24px
    *   Weight: 700 (Bold)
    *   Margin Bottom: 10px
*   **Rows:**
    *   Layout: Flexbox (Space Between) / Two Columns
    *   Font Size: 16px
    *   Padding: 0 10px (internal row padding)
    *   **Left Column (Day/Activity):** Weight 600 (Semi-Bold)
    *   **Right Column (Time):** Weight 400 (Normal)

### 4. Complex Timetables (Small Pool)
*   **Structure:**
    *   Standard Rows for Daytime.
    *   **Divider:** 1px dashed border with label "Evening Sessions" (Font 14px, Bold).
    *   Standard Rows for Evening times.

### 5. Special Display (Special Hours)
*   **Text:**
    *   Size: 48px
    *   Weight: 600 (Semi-Bold)
    *   Line Height: 1.2
    *   Content: Time range separated by a dash (e.g., 8:00 - 15:00)

## HTML Structure Reference

### Standard Text
```html
<p style="font-weight: 700; font-size: 24px; margin-bottom: 4px;">Title</p>
<p style="font-size: 20px;">Body Text</p>
```

### Two-Column Row
```html
<div style="display: flex; justify-content: space-between; padding: 0 10px; font-size: 16px;">
    <span style="font-weight: 600;">Left</span>
    <span>Right</span>
</div>
```
